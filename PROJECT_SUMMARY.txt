ÉTAPES DU PIPELINE BIG DATA :
────────────────────────────────────────────────────────────────────────────────

┌──────────────────────────────────────────────────────────────────────────────┐
│ PARTIE A : GÉNÉRATION DU DATASET (Obligatoire) - 4 points                    │
├──────────────────────────────────────────────────────────────────────────────┤
│ ✅ 1. Candidats.txt         → 5000 lignes, format : Num_CIN Nom Prenom Filière │
│ ✅ 2. Notes.txt             → 5000 lignes, 8 notes par candidat (0-20)      │
│ ✅ 3. Choix.txt             → 5000 lignes, 3 choix par candidat            │
│ ✅ 4. filiere.json          → Dictionnaire {code: [école, capacité]}      │
└──────────────────────────────────────────────────────────────────────────────┘
           ↓
┌──────────────────────────────────────────────────────────────────────────────┐
│ PARTIE B : STOCKAGE HDFS - 2 points                                         │
├──────────────────────────────────────────────────────────────────────────────┤
│ ✅ Créer /orientation dans HDFS                                             │
│ ✅ Importer tous les fichiers du dataset                                   │
│ ✅ Vérifier la structure avec : hdfs dfs -ls /orientation/                │
└──────────────────────────────────────────────────────────────────────────────┘
           ↓
┌──────────────────────────────────────────────────────────────────────────────┐
│ PARTIE C : MAPREDUCE - SCORE FG & RANG - 4 points                          │
├──────────────────────────────────────────────────────────────────────────────┤
│ ✅ MAPPER   : Fusionne candidats.txt + notes.txt                            │
│ ✅ REDUCER  : Calcule FG pour chaque candidat                              │
│ ✅ Formule  : FG = Analyse*8 + Algèbre*6 + ... + Anglais*3                │
│ ✅ OUTPUT   : rang.txt avec [Num_CIN, FG, Rang] - TRIÉ ↓                 │
└──────────────────────────────────────────────────────────────────────────────┘
           ↓
┌──────────────────────────────────────────────────────────────────────────────┐
│ PARTIE D : HIVE - ANALYSES ANALYTIQUES - 3 points                           │
├──────────────────────────────────────────────────────────────────────────────┤
│ ✅ Créer 5 tables externes : candidats, notes, rang, choix, resultats    │
│ ✅ Requête 1 : SELECT matière, AVG(note) GROUP BY matière                 │
│ ✅ Requête 2 : SELECT AVG(FG), MIN(FG), MAX(FG)                           │
│ ✅ Requête 3 : SELECT filiere, COUNT(*) GROUP BY filiere (MP vs PC)      │
│ ✅ Requête 4 : SELECT * ORDER BY FG DESC LIMIT 10 (TOP 10)              │
│ ✅ Requête 5 : SELECT filiere_affectee, COUNT(*)                        │
└──────────────────────────────────────────────────────────────────────────────┘
           ↓
┌──────────────────────────────────────────────────────────────────────────────┐
│ PARTIE E : HBASE - AFFECTATIONS NATIONALES - 3 points                       │
├──────────────────────────────────────────────────────────────────────────────┤
│ ✅ Table FILIERE   : Row=Code_Filière, Cols=capacité, places_restantes    │
│ ✅ Table CHOIX     : Row=Num_CIN, Cols=choix1, choix2, choix3            │
│ ✅ Table RESULTATS : Row=Num_CIN, Cols=filière_affectée, rang            │
│ ✅ Algorithme      : Pour chaque candidat (rang décroissant)              │
│    ├── Tester choix1, sinon choix2, sinon choix3                         │
│    ├── Décrémen capacité dans HBase                                      │
│    └── Générer resultats.txt                                            │
└──────────────────────────────────────────────────────────────────────────────┘
           ↓
